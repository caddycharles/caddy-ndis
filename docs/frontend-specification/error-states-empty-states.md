# Error States & Empty States

## Error States

### Network Error
```
┌─────────────────────────────────┐
│         ⚠️                      │
│   Connection Lost               │
│                                 │
│ We're having trouble connecting │
│ Check your internet and try    │
│ again.                         │
│                                 │
│ [Try Again]  [Work Offline]    │
└─────────────────────────────────┘
```

### Form Validation Error
```
┌─────────────────────────────────┐
│ ❌ Please fix these errors:     │
│                                 │
│ • NDIS number must be 9 digits │
│ • End date can't be in past    │
│ • Budget amount is required    │
└─────────────────────────────────┘
```

### 404 Not Found
```
┌─────────────────────────────────┐
│         🔍                      │
│   Page Not Found               │
│                                 │
│ The page you're looking for    │
│ doesn't exist or has been      │
│ moved.                         │
│                                 │
│ [Go to Dashboard]              │
└─────────────────────────────────┘
```
